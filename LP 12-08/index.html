Escrituras condicionais

Condicional simples

Estruturas condicionais servem para alterar o fluxo de exercução do meu algoritmo. Após um teste de condição, se a condição for verdadeira, o fluxo de exercução do codigo interno a estrutural condicional executa. Caso contario, o codigo salt para o final da estrutura condicional.

if(teste de condição){ comandos executados quando a condição é verdadeira }

Condicional Composta

if(teste de condição){ comando exercutados quando a condição é verdadeira } else{ comandos executados quando a condição é falsa }
